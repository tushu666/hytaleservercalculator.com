# Hytale 服务器配置推荐网站 - 设计思路

## 选定设计方案：**技术极简主义 + 数据驱动设计**

### 设计哲学
这个网站的核心目标是帮助玩家快速、准确地找到适合自己的服务器配置。设计应该围绕**信息清晰度**和**决策效率**展开，而不是过度装饰。采用技术极简主义风格，强调数据可视化和交互的直观性。

### 核心设计原则

1. **数据优先**：所有视觉元素都为数据展示服务。图表、对比表格、成本计算器是核心，不是附加品。
2. **渐进式信息架构**：从简单的推荐开始，逐步深入到成本计算和性能分析，让用户按需探索。
3. **高对比度与清晰度**：使用深色背景配合高饱和度的强调色，确保数据和交互元素在视觉上突出。
4. **功能即美学**：交互应该流畅、反馈即时，动画服务于理解而非装饰。

### 色彩哲学

- **主色**：深蓝色 (`oklch(0.35 0.15 250)`) - 代表技术、可靠性和专业感
- **强调色**：青绿色 (`oklch(0.65 0.15 180)`) - 用于 CTA 按钮、数据高亮、成功状态
- **背景**：深灰黑 (`oklch(0.12 0.01 0)`) - 减少眼睛疲劳，突出内容
- **文本**：浅灰白 (`oklch(0.92 0.01 0)`) - 高对比度，易于阅读
- **辅助色**：橙红色 (`oklch(0.65 0.2 25)`) - 用于警告、高成本提示

### 布局范式

采用**非对称网格布局**：
- 顶部：导航 + 品牌区域（简洁、固定）
- 主区域：左侧推荐卡片（宽度 40%），右侧成本估算器和图表（宽度 60%）
- 底部：性能对比图表（全宽）
- 移动端：堆叠式布局，优先级递减

### 签名元素

1. **配置卡片**：带有渐变边框和微妙阴影的卡片，表示不同的服务器配置等级
2. **数据可视化**：使用 Recharts 生成的图表，配合自定义颜色和平滑动画
3. **成本估算器**：实时计算反馈，输入框下方显示动态成本条形图

### 交互哲学

- **即时反馈**：用户输入时立即更新成本计算和图表
- **悬停增强**：卡片和图表元素在悬停时显示详细信息（tooltip）
- **平滑过渡**：所有状态变化都伴随 300-500ms 的过渡动画
- **键盘可访问性**：所有交互元素都支持键盘导航

### 动画指南

- **入场动画**：卡片从下往上淡入，间隔 100ms；图表数据从 0 动画到最终值
- **悬停状态**：卡片轻微上浮（transform: translateY(-4px)），阴影加深
- **加载状态**：脉冲动画，表示数据正在计算
- **过渡效果**：所有颜色、尺寸变化都使用 cubic-bezier(0.4, 0, 0.2, 1) 缓动

### 排版系统

- **显示字体**：`Sora` (Google Fonts) - 现代、几何感强、适合技术产品
- **正文字体**：`Inter` - 清晰、中性、易于阅读
- **字体层级**：
  - 页面标题：`Sora 700 text-4xl` - 品牌感
  - 卡片标题：`Sora 600 text-xl` - 配置名称
  - 数据标签：`Inter 500 text-sm` - 技术参数
  - 正文：`Inter 400 text-base` - 描述文本

### 响应式设计

- **桌面** (1024px+)：双列布局，推荐卡片和计算器并行
- **平板** (768px-1023px)：单列布局，卡片全宽
- **手机** (< 768px)：堆叠式，优化触摸交互
